📋 使用说明 - 审计2501班高数成绩查询系统
🎯 系统要求
操作系统：Windows 10 / Windows 11

内存：至少4GB

浏览器：Chrome、Edge、Firefox等现代浏览器

🚀 使用步骤
1. 下载和解压
下载 ccScoreSearch_发布包_YYYYMMDD_HHMM.zip 文件

右键点击ZIP文件 → 选择"解压到当前文件夹"

2. 运行系统
进入解压后的文件夹

双击运行 启动系统.bat 文件

3. 访问系统
等待命令行窗口显示启动信息

打开浏览器（Chrome/Edge等）

在地址栏输入：http://localhost:5000

按回车键访问

4. 查询成绩
在搜索框中输入学号或姓名

点击"查询"按钮查看成绩报告

可以查看班级统计、学习分析等信息

⚠️ 常见问题
如果无法启动：
防火墙警告：首次运行时，Windows防火墙可能弹出警告，请选择"允许访问"

端口占用：如果5000端口被占用，程序会自动提示

管理员权限：如果无法启动，尝试右键"以管理员身份运行"

如果无法访问网页：
确保程序已完全启动（命令行窗口显示服务地址）

检查浏览器地址是否正确：http://localhost:5000

尝试刷新页面或更换浏览器

🔧 故障排除
程序闪退

确保系统已安装 .NET Framework

尝试以管理员身份运行

无法连接

检查防火墙设置

确保5000端口未被其他程序占用

其他问题

重启电脑后重试

查看文件夹内的"使用说明.txt"
